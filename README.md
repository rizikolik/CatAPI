## Usage

<details><summary><b>Show instructions</b></summary>

1. Install the the app:

   ```sh
   $ bundle install
   ```

2) How to see project:

   ```sh
   $ bundle exec rails s
   go to localhost of your machine
   ```

</details>

3. Restful
4. Bootstrap
5. Font Awesome
6. Flash Messages for Notifications
7. Sass
8. Erb and Views in Rails
